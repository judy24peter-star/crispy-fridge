##Crispy-fridge
Smart Grocery Reminder
 simple AI tool that helps you remember when to buy groceries
 Summary
This project learns your shopping habits and tells you when you might need to restock items like milk, bread, or eggs. It uses basic pattern recognition and simple date calculations
Problem
People often forget to buy everyday items and end up running out at the wrong time. This tool helps avoid that by giving simple reminders based on your own routine.

How It Works
You enter an item and how often you buy it (example: “milk every 5 days”)

The system saves this information.

It calculates the next date you will likely need that item.

It shows a list of items you may need soon
